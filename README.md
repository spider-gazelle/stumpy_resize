# stumpy resize

Image resize algorithms in pure crystal

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     stumpy_resize:
       github: spider-gazelle/stumpy_resize
   ```

2. Run `shards install`
